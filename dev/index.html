Dodds_Watts_model.Agent — Type

Definition of the individual agents

type 1: Susceptible type 2: Infected type 3: Removed

dose: Count of received dosages

threshold: individual dose threshold; if the agents exceeds its threshold, it becomes infected

vaccine: Vaccinated yes or no

Dodds_Watts_model.plot_model_vacc — Method

Plots a simulation of the vaccination model for t timesteps, N individuals, S suscpetibles, I infected, R recoverd, exposure probability p, memory T, recovery rate r, probabiltiy imunity loss ρ, vaccination rate v and strength s The vaccination rate determines how many of the agents are being vaccinated. Their threshold will be raised by the vaccine strength s. The variable version determines the underlying model: "etm" = epidemic threshold model "ciritical" = critical mass model "universal" = heterogeneity model
